## Project Overview

Static personal portfolio/CV website for a cybersecurity professional. Modern, minimalist white-mode design. Pure HTML/CSS/JS with no frameworks or build tools. Hosted on GitHub Pages at taguirov.com.

## Development

No build step. Serve with any static HTTP server (e.g. `python3 -m http.server`).

## File Structure

```
index.html          — SPA shell (minimal static HTML)
css/style.css       — All styling (design tokens via CSS custom properties + components)
js/i18n.js          — Language detection, JSON loading, caching, language switch
js/renderer.js      — Pure rendering functions (data → HTML strings)
js/app.js           — Init, tab routing, event wiring, main render loop
data/en/*.json      — English content (9 files: profile, experiences, education, certificates, skills, projects, conferences, contact, ui)
data/fr/*.json      — French content (same 9 files)
assets/img/         — Profile photo, company logos, conference photos, project images
CNAME               — Custom domain config
```

## Architecture

- **Data-driven SPA**: All content lives in JSON files under `data/{lang}/`. HTML is generated by JS renderer functions. To add/edit content, only modify JSON files.
- **i18n**: EN default, FR available. Language preference saved in localStorage. Toggle button in header.
- **Tabs**: CV (default), Projects, Contact. Hash-based routing (#cv, #projects, #contact).
- **Global namespace**: `App` object with `App.i18n`, `App.renderer`, `App.state`.

## Deployment

Push to `main` — GitHub Pages auto-deploys. Custom domain `taguirov.com` via CNAME.

## Style Conventions

- Font: Outfit (Google Fonts), weight 300 for body, 200-400 for headings
- Color scheme: white background, near-black text (#212529), blue accent (#228BE6)
- All colors/spacing/typography defined as CSS custom properties in `:root`
- Responsive breakpoints: 768px, 640px, 380px
